# DevFlow AI Enterprise

An AI-powered developer productivity platform built using the MERN Stack and Google Gemini AI.

## Features

- User Authentication (JWT)
- Project Management Dashboard
- AI-Powered Developer Assistant
- Real-time Chat Interface
- Project Progress Tracking
- Secure Backend APIs
- MongoDB Atlas Integration
- Responsive UI

## Tech Stack

### Frontend
- React.js
- CSS3
- Axios

### Backend
- Node.js
- Express.js
- JWT Authentication

### Database
- MongoDB Atlas

### AI Integration
- Google Gemini AI
## Installation

```bash
npm install
cd client
npm install
```

---

## 🔑 Environment Variables

Create a `.env` file using `.env.example`

```env
PORT=5000
MONGO_URI=your_mongodb_uri
JWT_SECRET=your_jwt_secret
GEMINI_API_KEY=your_gemini_api_key
CLIENT_URL=http://localhost:3000
```

---

## ▶️ Run Application

```bash
npm run dev
```

Frontend:

```text
http://localhost:3000
```

Backend:

```text
http://localhost:5000
```
